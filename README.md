# Онлайн чат на C++ с использованием ASIO

Простой многопользовательский чат-сервер и клиент на C++ с использованием библиотеки **ASIO** (не boost). Поддерживает одновременное подключение нескольких клиентов, обмен сообщениями в реальном времени.

## Возможности

- Сервер может обслуживать несколько клиентов одновременно.
- Клиент подключается к серверу и отправляет сообщения.
- Сервер рассылает сообщения всем остальным подключённым клиентам.
- Протокол на основе TCP, порт по умолчанию: **45555**.
- Выход из чата по команде `exit` или через Ctrl+C.

## Требования

- Компилятор с поддержкой C++11 (или новее):  
  `g++`, `clang++`, MSVC
- CMake версии 3.5 или выше
- Стандартная библиотека ASIO (уже включена в репозиторий)

## Зависимости

В проекте используется **standalone ASIO** (версия 1.36.0). Все необходимые заголовочные файлы уже находятся в `libs/asio-1.36.0/include`. Дополнительная установка не требуется.


## Сборка

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

После успешной сборки в папке build/ появятся исполняемые файлы:
- OnlineChatserver – сервер (на Linux) или OnlineChatserver.exe (на Windows)
- OnlineChatclient – клиент (на Linux) или OnlineChatclient.exe (на Windows)
- Примечание: Если вы используете Windows, убедитесь, что CMake сконфигурирован на правильный генератор (например, -G "MinGW Makefiles" для MinGW).

## Запуск

### 1. Запуск сервера

По умолчанию сервер слушает порт `45555` на всех интерфейсах. Вы должны увидеть:

```
Сервер запущен на порту 45555
```

### 2. Запуск клиента

Клиент подключается к `127.0.0.1:45555`. При успешном подключении появится сообщение:

```
    Подключено к серверу!
    Для выхода введите 'exit' или нажмите Ctrl+C
```

## Использование

- Введите текст и нажмите Enter – сообщение отправится всем остальным клиентам.
- Сообщение отображается в формате:  
  `Клиент <ID>: <текст>`
- Для выхода из чата введите `exit` или нажмите Ctrl+C.

Пример сессии (два клиента):

**Клиент 1**  

```
    Подключено к серверу!
    Привет всем!
    Клиент 2: Привет!
```
**Клиент 2**  

```
    Подключено к серверу!
    Клиент 1: Привет всем!
    Привет!
```

## Настройка

В текущей версии IP-адрес сервера и порт захардкожены в коде клиента (`127.0.0.1:45555`) и сервера (`0.0.0.0:45555`). Чтобы изменить их, отредактируйте соответствующие строки:

- Клиент: `socket.connect(tcp::endpoint(asio::ip::make_address("127.0.0.1"), 45555));`
- Сервер: `tcp::acceptor acceptor(io_context, tcp::endpoint(tcp::v4(), 45555));`

## Возможные проблемы

### Устройства не находятся в локальной сети

Убедитесь, что сервер и клиенты действительно могут друг до друга достучаться, если запускаете чат в локальной сети!

### Порт уже занят
Если при запуске сервера вы видите ошибку `Address already in use`, измените порт в коде или завершите процесс, использующий порт 45555.

### Клиент не подключается
Убедитесь, что сервер запущен и использует тот же порт. Если сервер и клиент работают на разных машинах, укажите реальный IP-адрес сервера в коде клиента.

### Ошибка сборки с ASIO
ASIO требует наличия системы сборки для сетевых вызовов. На Linux необходимо линковать с `pthread`. В CMakeLists это не указано явно, но ASIO добавляет нужные флаги самостоятельно. Если возникнут проблемы, добавьте в `CMakeLists.txt`:

    target_link_libraries(${CN} pthread)
    target_link_libraries(${SN} pthread)
